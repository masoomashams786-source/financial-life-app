financial-life-app — Personal Financial Planning Platform

A full-stack financial intelligence platform that transforms raw financial data into actionable insights, multi-scenario projections, and side-by-side comparisons of investment vehicles — built for individuals who want clarity over their financial future.


 Overview
This is a modern web application that empowers users to track their financial health, plan for retirement, and compare investment strategies with confidence. Users input their financial snapshot (income, expenses, savings, investments, debt), and the platform produces 30-year net worth projections across optimistic, realistic, and pessimistic market scenarios — alongside a proprietary health score benchmarked against Federal Reserve data.
The app also features a Financial Vehicles Hub where users can manage their investment plans, explore a comparative matrix of 12+ financial instruments (Roth IRA, 401k, IUL, HSA, and more), and run a multi-plan calculator to model potential returns before committing capital.

## Features
 
### Authentication
- Secure JWT-based authentication flow
- Password hashing with Bcrypt
- Protected routes on both frontend and backend
- Auto-redirect for authenticated/unauthenticated users
 
### Personal Dashboard
- **Financial Snapshot Card** — real-time summary of income, expenses, savings, investments, and debt
- **Net Worth Projection Chart** — multi-scenario (best / predicted / worst) projections over 30+ years, powered by a custom `ProjectionEngine` using historical S&P 500 returns and inflation modeling
- **Financial Health Score** — algorithmic score benchmarked against Federal Reserve median data by age group, surfacing strengths and vulnerabilities
- **Insights Panel** — curated, actionable recommendations generated by the `InsightEngine`
 
###  Financial Vehicles Hub (3-tab interface)
| Tab | Description |
|---|---|
| **My Current Plans** | Full CRUD management of personal investment plans with contribution tracking and employer match support |
| **Comparative Matrix** | Side-by-side feature comparison of 12 financial instruments (IUL, Whole Life / IBC, Roth IRA, Traditional 401k, Roth 401k, Solo 401k, HSA, 529 Plan, Real Estate, Private Equity, CDs, Annuities) |
| **Plan Calculator** | Multi-plan projection calculator — configure up to N plans with custom parameters and compare projected outcomes; save winning scenarios directly to your plans |
 
### Projection Engine (Pure Algorithmic)
- Year-by-year net worth, cash flow, and milestone tracking
- Three market scenarios: Predicted (7%), Best (10%), Worst (3%)
- Income growth scenarios: Conservative (3%), Optimistic (5%), Pessimistic (-2%)
- Inflation-adjusted calculations at 3% historical average
- Projects from current age to configurable retirement age
 
### Insight Engine (Pure Algorithmic)
- Composite financial health score with weighted metric categories
- Benchmarked against Federal Reserve SCF data by age cohort
- Identifies financial strengths and vulnerabilities
- Generates prioritized, specific action recommendations
 
### News Integration
- Financial news feed surfaced in the dashboard

##  Getting Started
 
### Prerequisites
- Python 3.10+
- Node.js 18+
- A relational database (SQLite for development, PostgreSQL recommended for production)
 
### Backend Setup
 
```bash
# Clone the repository
git clone https://github.com/your-username/wealthpath.git
cd wealthpath/backend
 
# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
 
# Install dependencies
pip install -r requirements.txt
 
# Configure environment variables
cp .env.example .env
# Edit .env with your DATABASE_URL and JWT_SECRET_KEY
 
# Run database migrations
flask db upgrade
 
# Seed contribution limits (IRS data)
python seed_contribution_limits.py
 
# Start the development server
python run.py
```
 
### Frontend Setup
 
```bash
cd ../frontend
 
# Install dependencies
npm install
 
# Start the development server
npm run dev
```
 
The app will be available at `http://localhost:5173` with the API running on `http://localhost:5000`.

## Design Decisions
 
- **Pure algorithmic engines** — `ProjectionEngine` and `InsightEngine` are written as standalone, testable Python classes with no external AI/API dependencies, ensuring fast, deterministic, and free-to-run financial modeling.
- **SWR for data fetching** — automatic background revalidation keeps the dashboard fresh without manual refresh triggers.
- **JWT stateless auth** — no server-side session storage; scales horizontally without sticky sessions.
- **Marshmallow serialization** — strict input validation and output shaping at the API boundary prevents data inconsistencies.
- **Alembic migrations** — all schema changes are versioned, enabling safe rollbacks and team collaboration.
